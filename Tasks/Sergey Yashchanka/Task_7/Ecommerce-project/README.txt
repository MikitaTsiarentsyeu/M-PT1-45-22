ЗАДАНИЕ:
# Реализовать каталог товаров, основанный на шаблоне 3-х слойной архитектуры.
# Пользователь должен иметь возможность просматривать товары из определённой категории, добавлять
# определённое количество товаров в корзину и делать окончательный заказ.
# Минимальный набор характеристик товара: название, категория, товарный код, цена, (количество на складе).

----------------------------------------------------------------------------------------------------------------
program version 1.0 !

Интернет магазин по продаже электронной техники!

Программа загружает данные из базы данных, которой служит 'json' файл и после оплаты пользователем товара
сохраняет обратно в json обновленные данные! В этой программе пользователь может просматривать все товары
по всем категориям, может просматривать товары по отдельным категориям, добавлять их в корзину и делать
окончательный заказ. А так же программа считает стоимость товаров на случай если пользователь хочет приобрести
несколько одинаковых товаров и показывает их стоимость.
